# OGTOP - System Monitor

Linux sistemler için Python ile geliştirildi, estetik ve detaylı bir terminal tabanlı sistem izleme aracı. 

## Özellikler
- **CPU İzleme:** Her çekirdek için canlı grafik barları.
- **GPU Bilgisi:** NVIDIA ekran kartları için kullanım ve sıcaklık takibi.
- **RAM & Disk:** Doluluk oranları ve görsel göstergeler.
- **Top 5 Süreç:** Sistemi en çok yoran 5 uygulama ve anlık sonlandırma (Kill) özelliği.
- **Ağ Trafiği:** İndirme ve yükleme hızları.

## Kurulum

Terminali açın ve proje klasöründe şu komutları sırayla çalıştırın:

```bash
chmod +x install.sh
./install.sh
```

Kurulum bittikten sonra yeni terminal açabilir veya source ~/.bashrc yazarak hemen ogtop komutunu kullanmaya başlayabilirsiniz.
