# OGTOP - System Monitor

Linux sistemler için Python ile geliştirildi, estetik ve detaylı bir terminal tabanlı sistem izleme aracı.

## Özellikler
* **CPU İzleme:** Sıcaklık takibi ve dinamik durum göstergesi.
* **GPU Bilgisi:** NVIDIA kullanım ve sıcaklık verileri.
* **Dinamik Durum:** Sıcaklığa göre değişen COOL/STABLE/HOT uyarıları.
* **Top Processes:** En çok kaynak tüketen 5 uygulama.

## Kurulum
```bash
python3 monitor.py
